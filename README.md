# ax-boot-html

본 저장소는 https://github.com/axboot/ax-boot-framework의 프론트앤드 부분만을 별도로 정리해둔 저장소입니다.

## 실행방법
터미널 창에 다음의 커멘드를 순서대로 입력하세요. npm 명령어가 실행되지 않는다면 nodejs를 설치하셔야 합니다.
``` sh
npm install
npm start 
```

npm start(node app.js) 명령어 실행 후 브라우저에서 확인 <http://localhost:4000/>
